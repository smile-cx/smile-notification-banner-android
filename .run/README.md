# Android Studio Run Configurations

This directory contains shared run configurations for Android Studio.

## Available Configurations

- **sample** - Runs the sample application that demonstrates the SmileBanner library

## How to Use

1. **Open the project in Android Studio from the ROOT directory** (not the `sample` subdirectory)
2. The run configurations will be automatically available in the "Run" dropdown
3. Select "sample" from the dropdown and click Run

## Important Notes

- Always open the project from the root directory of this repository
- Do NOT open the `sample` directory as a separate project
- If you see errors about missing tasks, close Android Studio and reopen from the root directory
